# 🎮 Pac-Man Game

一個使用純 HTML5 Canvas 和 JavaScript 開發的經典 Pac-Man 遊戲。

## 🎯 遊戲特色

- 🟡 經典的 Pac-Man 角色與動畫
- 👻 四個不同顏色的幽靈 AI
- 🔵 經典迷宮設計
- 🏆 分數系統
- 🎮 流暢的遊戲體驗
- 📱 響應式設計

## 🎮 如何遊玩

- 使用方向鍵（↑↓←→）控制 Pac-Man 移動
- 收集黃色小豆子獲得 10 分
- 收集大能量豆獲得 50 分
- 避免被幽靈抓到
- 收集完所有豆子即可獲勝

## 🚀 快速開始

1. 下載或克隆此項目
2. 在瀏覽器中打開 `index.html`
3. 開始遊戲！

這個遊戲完全使用原生 Web 技術開發，無需任何外部依賴。遊戲邏輯包括：
